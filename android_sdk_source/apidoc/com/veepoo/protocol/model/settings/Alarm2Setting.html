    data of new alarm clock, two types of alarm clocks can be set,

    A.Repeated state alarm [specify Monday, Tuesday,...],This alarm clock will be set on every Monday, every Tuesday, every week..will remind the alarm clock, this alarm clock needs to be set whilerepeatStatusproperty needs to be set,unRepeatDateProperties 0000-00-00

    B.Non-repeating alarm clock „Äê2017-04-10], this alarm clock will only remind the alarm on the day unRepeatDateproperties, whilerepeatStatusattribute needs to be set to 0000000
